# Lawyers Book - دليل المحامين ⚖️

A modern, fast, and easy-to-use desktop application for searching and managing the Palestinian Bar Association's lawyers directory. 

![Application Icon](app_icon.png)

## Features

- 🔍 **Instant Search**: Search through 4700+ records by Name, City, or Membership ID in real-time.
- ➕ **Add New Lawyers**: Easily add new entries to the local database through a simple interface.
- 📋 **Copy-Paste Support**: Select and copy any contact information directly from the details pane.
- 🌍 **Arabic Language Support**: Full RTL (Right-to-Left) support for Arabic text and data.
- 🎨 **Modern Interface**: Professional "Bootstrap" style UI built with `ttkbootstrap`.
- 📦 **No Installation Required**: Packaged as a standalone EXE.

## How to Use

1. **Download**: Grab the latest `Lawyers Book.zip` from the [Releases](https://github.com/samakeh/lawyers-numbers/releases) page.
2. **Extract**: Unzip the folder to your preferred location.
3. **Run**: Open the folder and double-click `Lawyers Book.exe`.

## Technical Setup (For Developers)

If you want to run the source code or build it yourself:

1. Clone the repository.
2. Create a virtual environment: `python -m venv venv`.
3. Install dependencies: `pip install -r requirements.txt`. (Dependencies include `ttkbootstrap`, `openpyxl`, `pandas`, `pillow`, `pyinstaller`).
4. Run the app: `python main.py`.

## Data Source
The initial data is sourced from the Palestinian Bar Association's official directory.
